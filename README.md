# CATWAALA - Cat Welfare Bangladesh

A comprehensive cat welfare platform for rescuing, rehabilitating, and rehoming stray cats in Bangladesh.

## About

Catwaala is a non-profit animal welfare organization with a 450K+ Facebook community, dedicated to helping stray cats across Bangladesh find loving forever homes.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fmyselfrezaul7%2FCatwaala---Animal-welfare)

Before deploying, make sure to update the `.env` file with your Firebase and reCAPTCHA credentials.
